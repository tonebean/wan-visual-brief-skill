# 品牌 KV 定调助手 — Demo 文档

本文件用于展示本 Skill 在同一个产品上，如何应对不同品牌任务。

这次三组 Demo 不再用三个不同产品，而是统一锁定为同一个产品锚点：

**同一支高端科技护肤精华瓶：透明圆柱玻璃瓶，银色短颈金属泵头，瓶内一根细长滴管造型，冷白精华液，极简无装饰。**

这样更能看出这套 skill 真正在做什么：

- 不是靠换产品制造差异
- 而是靠任务判断、路线选择和后续校正，拉开同一产品在不同 KV 任务下的结果差异

每个 Demo 都统一展示以下内容：

1. 原始用户 brief
2. Brief 解读
3. 视觉路线生成
4. 推荐路线
5. 执行描述
6. 最终 Wan Prompt
7. 对应价值说明
8. 下一步判断

当前演示图统一放在 `examples/` 目录。

---

# Demo 1：同一产品的品牌定调主视觉

## 当前演示图

![Demo 1](examples/demo1-main-kv.png)

## 原始用户输入

> 用同一支精华瓶做品牌定调主视觉，要高级、科技感强一点，有大牌感。

## 如果直接拿原话去生图，常见问题

- 容易只剩“科技感”和“高级感”这种空词
- 容易做成普通产品图，品牌气质不稳
- 容易冷感过头，像实验概念图，不像真正的品牌主视觉

## Brief 解读

### 任务识别
这不是单纯卖产品功能的图，首先承担的是 **品牌调性建立 + 高端感传递**，其次才是产品展示。

### 第一感受目标
观众第一眼应该感受到：
- 精密
- 洁净
- 昂贵
- 值得信任

### 视觉任务
这张图的工作不是讲很多卖点，而是先建立一种：

**“这是一支高科技、高质感、高价位护肤精华”** 的印象。

## 视觉路线生成

### 路线 A：冷感科技精密型
- 核心气质：冷静、洁净、理性、高科技实验室感
- 视觉逻辑：通过玻璃、液体、冷白光、精密构图和实验室空间，建立科技护肤的可信度
- 适用原因：适合强调品牌科技力和高端感，更像国际护肤品牌主视觉
- 不适合什么：不适合需要明显温度感或柔和奢华氛围的 brief；容易做得太冷

### 路线 B：高奢材质质感型
- 核心气质：昂贵、克制、珠宝感、奢华但不过度炫耀
- 视觉逻辑：通过高级材质、细腻反光和大面积留白，建立高价位产品的精致质感
- 适用原因：适合偏品牌形象和高端种草，更像大牌 campaign KV
- 不适合什么：不适合用户明确要求强科技感的 brief；容易只剩“贵”，但科技可信度不够

### 路线 C：流体微观生命型
- 核心气质：柔和、流动、微观、生命感
- 视觉逻辑：通过液体、肌理和成分世界建立“有效成分正在工作”的想象空间
- 适用原因：适合更年轻、更内容传播向的视觉表达
- 不适合什么：不适合做正式品牌定调主视觉；容易更像成分传播图

## 推荐路线

### 推荐：路线 A「冷感科技精密型」

推荐原因：
因为这个任务最重要的是：
- 高级
- 科技感强
- 大牌感

路线 A 更能同时满足“科技感”和“品牌定调”。

为什么不推荐另外两条：
- 不选路线 B：它能做出昂贵感，但对“科技感强一点”的回应不够硬
- 不选路线 C：它更适合内容传播，不够像正式品牌 KV

## 执行描述

- 主体：同一支高端科技护肤精华瓶作为唯一主角
- 构图：稳定中轴构图，大面积冷白留白
- 景别 / 镜头感：中近景产品 hero shot
- 光线：冷白主光与克制边缘光
- 色彩系统：冷白、银灰、低饱和冰蓝
- 材质表现：玻璃、液体、抛光金属
- 环境：抽象实验室空间
- 比例建议：竖版主视觉

## 最终 Wan Prompt

```text
同一支高端科技护肤精华瓶，透明圆柱玻璃瓶，银色短颈金属泵头，瓶内一根细长滴管造型，冷白精华液，极简无装饰，作为唯一主角，品牌定调主视觉，稳定中轴构图，大面积冷白留白，冷白主光与克制边缘光，玻璃与金属细节清晰，透明液体与抽象实验室空间营造高科技护肤氛围，整体洁净、昂贵、可信，品牌广告质感，真实商业摄影，竖版主视觉
```

## 这个 Demo 证明了什么

- Skill 能处理品牌调性和“高级感”这类抽象需求
- Skill 能把“科技感”翻译成材质、光线和空间语言
- 同一产品在品牌定调任务下，可以稳定做出主视觉气质

## 下一步判断

- 当前结论：可以直接交付，不需要再大改路线
- 如果还要继续修：优先只改瓶身局部反光、金属边缘精度或背景干净度
- 不建议怎么改：不要加人物或成分爆炸元素，不然会把它从品牌主视觉拉回传播图

# Demo 2：同一产品的新品发布主视觉

## 当前演示图

![Demo 2](examples/demo2-main-kv.png)

## 原始用户输入

> 还是这支精华瓶，做新品发布主视觉，要正式发布感、产品辨识度高，像官网 hero。

## 如果直接拿原话去生图，常见问题

- 容易做成普通产品棚拍图
- 容易只有产品居中，没有发布感
- 容易像电商图，不像正式品牌新品发布主视觉

## Brief 解读

### 任务识别
这张图不再以“品牌定调”为第一目标，而是更偏 **新品发布 + 产品辨识度建立**。

### 第一感受目标
观众第一眼应该感受到：
- 新
- 准
- 稳
- 像正式发布

### 视觉任务
这张图既要让同一支精华瓶看起来更有发布感，也要保持高端品牌气质，不能滑成普通产品页图片。

## 视觉路线生成

### 路线 A：发布空间通道型
- 核心气质：正式、前卫、发布感强、产品中心明确
- 视觉逻辑：通过克制的未来感空间、导向光线和更强产品轮廓建立新品发布氛围
- 适用原因：最适合“官网 hero”“正式发布主视觉”这种场景
- 不适合什么：不适合需要更柔和品牌情绪的 brief；容易太硬，少一点品牌温度

### 路线 B：极简雕塑展陈型
- 核心气质：静、贵、极简、像高级展览
- 视觉逻辑：让同一产品成为空间里唯一雕塑主角，靠留白和比例建立高端感
- 适用原因：适合做极简品牌官网首屏
- 不适合什么：不适合要求明显“新品发布感”的 brief；容易过静

### 路线 C：卖点传播内容型
- 核心气质：清楚、直接、卖点明确
- 视觉逻辑：通过更多说明性元素和辅助结构去强化产品信息
- 适用原因：适合后续做传播向内容图
- 不适合什么：不适合当前这个正式品牌主视觉任务；容易掉成内容图

## 推荐路线

### 推荐：路线 A「发布空间通道型」

推荐原因：
因为这次的核心不是单纯“好看”，而是：
- 正式发布感
- 产品辨识度高
- 官网 hero 气质

为什么不推荐另外两条：
- 不选路线 B：它够高级，但发布感不够明确
- 不选路线 C：它更适合后续卖点传播，不够像正式发布主视觉

## 执行描述

- 主体：同一支高端科技护肤精华瓶作为唯一主角
- 构图：横版官网 hero 构图，产品占画面主要面积
- 景别 / 镜头感：略低角度近景
- 光线：克制的导向光线，强调产品轮廓
- 色彩系统：冷灰、银白、低饱和未来感光线
- 材质表现：玻璃、金属、冷白液体
- 环境：克制的未来感发布空间与少量光带
- 比例建议：横版优先

## 最终 Wan Prompt

```text
同一支高端科技护肤精华瓶，透明圆柱玻璃瓶，银色短颈金属泵头，瓶内一根细长滴管造型，冷白精华液，极简无装饰，同一产品作为唯一主角，新品发布主视觉，横版官网 hero 构图，产品占画面主要面积，镜头略低角度，拉开产品轮廓，背景是克制的未来感发布空间与少量光带，不出现人物，不出现第二产品，强调正式品牌新品发布、产品辨识度、真实材质、发布会级别主视觉
```

## 这个 Demo 证明了什么

- Skill 不只是会给同一个产品做品牌定调
- 同一个产品进入“新品发布”任务后，能稳定切到更强的产品中心表达
- 差异来自任务判断和路线选择，不是来自换产品

## 下一步判断

- 当前结论：可以交付，也适合做很轻的局部细改
- 如果还要继续修：优先只改瓶身边缘光、发布空间的秩序感和背景杂光
- 不建议怎么改：不要补人物、生活场景或太多辅助元素，不然会把它拉回内容图

## Demo 3（延展案例）：同一产品的品牌概念延展图

## 当前演示图

![Demo 3](examples/demo3-extension-kv.png)

这一组放在最后，只用于说明：  
这套 skill 在同一产品上，也能从正式主视觉再延展到品牌概念图，但这不是它的主打方向。

原始输入：

> 还是这支精华瓶，做一张品牌概念延展图，要冷静、克制、有一点情绪，但不能像故事画面。

如果直接拿原话去生图，常见问题：

- 容易只剩夜景氛围，没有品牌线索
- 容易拍成故事感场景，而不是品牌概念图
- 容易让同一产品退成陪体

这组最终选择了「品牌装置空间型」路线，核心不是讲故事，  
而是让同一支精华瓶继续做前景主角，再用极简空间、冷暖色温和湿润反射地面建立更像 campaign teaser 的品牌概念感。

为什么不继续往“叙事情绪型”走：
- 那样更容易做成故事画面，而不是品牌概念图
- 一旦加入生活道具或明显动作，产品就会退成陪体
- 对这套 skill 来说，延展可以有，但不能把延展案例做成剧情图

这组的意义不是证明它适合剧情流程，  
而是说明它在同一产品上，也能从主视觉延展到更情绪化的品牌概念表达。

## 下一步判断

- 当前结论：适合做轻量局部校正，不适合再往叙事方向扩展
- 如果还要继续修：优先只改产品存在感、空间留白和冷暖光比例
- 不建议怎么改：不要补人物剧情、不要加生活道具、不要让故事感压过产品

# Demo 总结

这三组 Demo 现在统一验证的是：

1. **同一个产品，也能根据不同任务得到明显不同的 KV 结果**
2. **Skill 的差异来自任务判断、路线选择和后续校正，不来自换产品**
3. **这套东西更像品牌主视觉判断工具，而不是一次性生图器**

它们共同证明：

- 本 Skill 不是一次性 prompt 工具
- 本 Skill 具备可复用、可调用、可执行的工作流结构
- 同一产品在品牌定调、正式发布和概念延展这三种任务里，可以被清楚地区分开来
